# ShipReceipt v0.1.0

ShipReceipt creates and verifies tamper-evident delivery receipts for local file
bundles.

## Added

- Directory scanning with streaming SHA-256 hashing.
- Deterministic JSON receipt IDs.
- Verification for changed, missing, and added files.
- Optional HMAC-SHA256 signing and verification.
- Local signing key generation.
- CLI commands for create, verify, inspect, and keygen.
- Unit, integration, and E2E test coverage.

## Install

```console
python -m pip install shipreceipt-0.1.0-py3-none-any.whl
```

Or run the zipapp:

```console
python shipreceipt.pyz --version
```

## Use

```console
shipreceipt keygen signing.key
shipreceipt create ./bundle --output bundle.receipt.json --key signing.key
shipreceipt verify bundle.receipt.json --root ./bundle --key signing.key
```

## Known Limitations

- Receipts are JSON files and do not encrypt file names or sizes.
- v0.1.0 uses shared-key HMAC signing only.
- Verification reports are human-readable CLI output; structured report export is planned.
